# Procedural C vs OOP TaskList

## Problems in Procedural C Version

1. Global variables are used, making data management difficult.

2. Task data and related functions are separate, reducing code organization.

3. Adding new features requires modifying multiple parts of the program.

## How OOP Solves These Problems

1. Encapsulation keeps data and methods together inside classes.

2. Objects make code more organized and easier to understand.

3. Classes are reusable and easy to extend with new features.
